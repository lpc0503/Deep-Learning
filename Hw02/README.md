# Deep-Learning

## 注意事項

執行結束後會在`output`資料夾底下產生各 case `.txt`的輸出檔 輸出以及繪製的圖都在裡面

每次執行`caseX.py`都會隨機產生weight 和 learning rate

# #跟目錄下的檔案

* `data`
    * 訓練資料跟測試資料
* `output`
    * 輸出檔案放置的地方
* `src`
    * 原始碼
* `README.md`
    * 此文件
* `run.bat`
    * 執行腳本

## 執行方法

* 執行方法一
    * 執行 `run.bat`
        * 他會將所有 `case` 都執行 (case4 會跑出 warning 屬於正常情況)

* 執行方法二
    * 進入`src`資料夾並開啟 `cmd` 或其他 `ide` 並執行各 `caseX.py`
